# PRODIGY_DS_03

Task 3: Customer Purchase Prediction using Decision Tree Classifier

Objective

Build a Decision Tree Classifier to predict whether a customer will purchase a product or service based on demographic and behavioral data.

Dataset

Bank Marketing Dataset (UCI Machine Learning Repository)

Tools Used
	•	Python
	•	Pandas
	•	Scikit-learn
	•	Matplotlib
	•	Jupyter Notebook

Tasks Performed
	•	Loaded and explored the dataset
	•	Performed data preprocessing
	•	Selected relevant features
	•	Trained a Decision Tree Classifier
	•	Visualized the decision tree
	•	Evaluated the model’s performance

Outcome

Successfully developed a Decision Tree model capable of predicting customer purchase behavior based on demographic and marketing-related features.

Learning
	•	Data preprocessing for machine learning
	•	Decision Tree Classification
	•	Model training and evaluation
	•	Data visualization
	•	Predictive analytics using Python
